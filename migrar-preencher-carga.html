🔧 Preencher Carga Nominal

Usar dados originais do extintoresInfo para preencher campos corretos

⚠️ O que este script faz:

• Usa dados originais de extintoresInfo
• Preenche carga_nominal_valor e carga_nominal_unidade
• Detecta automaticamente unidade correta (kg ou L) pelo tipo
• Atualiza todos os 97 extintores

Execute para corrigir dados!
